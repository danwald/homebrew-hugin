# Danwald Hugin

## How do I install these formulae?

`brew install danwald/hugin/<formula>`

Or `brew tap danwald/hugin` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "danwald/hugin"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
